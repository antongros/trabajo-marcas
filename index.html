Tutorial de HTML

¿Ques es el HTML?


El HTML (HyperText Markup Language)es el lenguaje con el que se escriben las páginas web. Es un lenguaje que permite escribir texto de forma estructurada, y que está compuesto por etiquetas, que marcan el inicio y el fin de cada elemento del documento.
Los documentos HTML deben tener la extensión html o htm, para que puedan ser visualizados en los navegadores. Los navegadores se encargan de interpretar el código HTML de los documentos, y de mostrar a el usuario la página web resultantes del código.

Estructura básica de HTML


El modelo básico se HTML se compone de :
<!Doctype> Esta etiqueta declara que tipo de documento vamos a crear, siempre sera la primera etiqueta de nuestro documento.
<html>Esta etiqueta declara que a partir de aqui todo lo que escribamos sera HTML.
<head>Esta etiqueta crea una zona en nuestro documento llamada cabecera.
<title>Esta etiqueta se usa para poner un titulo a nuestra página web. Por ejemplo:
<title> Este es el título de mi web <\title>
<\head> Ahora cerrariamos la etiqueta HEAD
<body> Esta etiqueta abre la parte cuerpo de nuestra web en la que escribiremos todo el contenido.
<\body> Esta etiqueta cierra la etiqueta BODY.
<\html> Esta etiqueta cierra la etiqueta HTML abierta anteriormente.

Las listas ordenadas disponen de una serie de atributos:
  • reversed="": Invierte la numeración.
  • type="": Sirve para seleccionar el carácter que queremos usar para numerar la lista.
  • value="": Asigna el numero en que comenzara la lista.

Creacción de tablas

Las tablas se crean con la etiqueta <table></table>. Dentro de la etiqueta usamos las siguientes etiquetas:
  • <tr></tr>:Para crear una nueva fila a la que añadiremos columnas.
  • <td></td>:Para crear las columnas que necesitemos dentro de una fila.
  • <th></th>:Para crear la columna de encabezado.

  • rowspan="" : Es un atributo de <th> y <td>, ocupa el número de columnas hacia la derecha que se especifique.
  • colspan="" : Es un atributo de <th> y <td>, ocupa el número de filas hacia abajo que se especifique.
Además <table></table> tiene el atributo border="" para añadir el tamaño del borde de la tabla.

Mapas de imagen

        
		Se pueden delimitar zonas de una imagen para que te lleven a un enlace. Para ello se usa la etiqueta <map></map> y la etiqueta <area> para definir cada zona de la imagen. Además son necesarios los atributos que se mencionan a continuación:
				
				
    
					
  • shape="": Es el tipo de área que queremos crear (default, rect, circle, poly).
					
    • 
					
  • coords="": Son las coordenadas para el tamaño del área.
					
    • 
					
  • href="": Determina la dirección a la que hacemos referencia con la zona.
					
    • 
					
  • alt="": Comentario descriptivo de la zona (Obligatorio si usamos "href").
					
    • 
					
  • download: Descarga el fichero del enlace.
					
    • 
					
  • tarjet="": Permite elegir como abrir el enlace.
					
    • 
				

				
					Por ejemplo, vamos a crear una zona en la parte superior derecha  de la foto http://www.heraldo.es/:
					

				    <img src="lacasablanca.jpg" alt="La casa blanca" usemap="#heraldo">
				    <map name="Heraldo">
					<area shape="circle" coords="325,325,35" href="http://www.heraldo.es/" alt="el heraldo">
				    </map>
